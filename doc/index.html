DNI_NIE

A gem to validate Spanish Identity Documents rubygems.org/gems/ci_uy

Install:

$ gem install dni_nie

Usage: ruby fernando@hoth ~ $ rake console 1.9.3-p484 :001 > CiUY.validate("1.111.111-1") => true 1.9.3-p484 :002 > CiUY.validation_digit("2222222") => "2" 1.9.3-p484 :003 > CiUY.random => "13082956"

Once the gem is installed in your system, you can also use 'ci_uy' from the command line:

$ gem install ci_uy-0.0.6.gem
